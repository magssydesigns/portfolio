# portfolio

Personal portfolio showcasing product design case studies, AI-assisted workflows, and implementation-aware design.

Built with Next.js (App Router), TypeScript, Tailwind CSS, and Framer Motion.

## Getting started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view it locally.

## Structure

- `src/app` — pages: homepage, `/work` (listing), `/work/[slug]` (case study template), `/about`, `/ai-workflow`
- `src/components` — shared UI (nav, footer, motion primitives, case study cards)
- `src/lib/case-studies.ts` — case study content and types

## Content

Case study copy, metrics, and the bio in `/about` are placeholder content written for this template — replace with your own project details before publishing.
